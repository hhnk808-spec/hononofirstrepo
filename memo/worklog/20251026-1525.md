### 2. tasizan.py の作成と改良
- **ファイル**: `src/tasizan.py`
- **目的**: 1から50まで順に足していって累計を表示するプログラム

#### 初期バージョン
- 数式形式での表示（例: `1+2=3`）

#### 第1回改良
- 日本語形式に変更（例: `1足す2は3です。`）

#### 第2回改良（最終版）
- 関西弁形式に変更（例: `1足す2は3やで`）
- **最終コード**:
  ```python
  total = 0
  for i in range(1, 51):
      if i == 1:
          print(f"{i}やで")
      else:
          print(f"{total}足す{i}は{total + i}やで")
      total += i
  ```

## 実行方法

```bash
# tasizan.py の実行
python src/tasizan.py
```
